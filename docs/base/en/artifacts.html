This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Structures: Resource Profiles

These define constraints on FHIR resources for systems conforming to this implementation guide.

LT Base Condition

Lithuanian Base Condition profile, used for documenting general, symptoms, conditions, and their characteristics.
LT Base Encounter

Lithuanian Base Encounter profile, used to represent clinical encounters
LT Base Observation

Lithuanian Base Observation profile, used to represent vitals, demographic, clinical and laboratory observations
LT Base Organization

Lithuanian Base Organization profile, used to define healthcare organizations and sub-units
LT Base Patient

Lithuanian Base Patient profile, used to represent patients administrative information
LT Base Practitioner

Lithuanian Base Practitioner profile, used to define healthcare practitioners
LT Base Practitioner Role

Lithuanian Base Practitioner Role profile, used to define roles of the healthcare practitioner in the healthcare facilities
LT Base Related Person

Lithuanian Base Related Person profile, used to define persons related to a patient
